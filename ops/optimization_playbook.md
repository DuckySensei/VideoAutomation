# Optimization Playbook (Break-Even Focus)

## Objective
Shrink the break-even gap every 2 weeks by improving retention and reducing wasted production.

## Biweekly Rules
- Keep top 20-30% formats by:
  - retention,
  - average view duration,
  - revenue contribution.
- Pause formats with low retention for 2 consecutive weeks.
- Replace one weak format with a new angle in the same niche.

## Cost Controls
- Set hard monthly API caps.
- Reuse approved templates and avoid unnecessary generation volume.
- Batch render jobs once daily instead of ad-hoc runs.

## Revenue Uplift Actions
- Improve first line hook in scripts.
- Tighten titles to one specific promise.
- Add one clear conversion CTA per video (affiliate click, newsletter, tool link).

## Break-Even Targets
- If monthly fixed cost is `$30`, target `>= $1/day` by month 2-3.
- If monthly fixed cost is `$50`, target `>= $1.70/day` by month 3-4.
- If monthly fixed cost is `$75`, target `>= $2.50/day` by month 4-6.

## Weekly Dashboard Checklist
- Revenue this week
- Variable costs this week
- Fixed cost allocation this week
- Gap to break-even
- Top 3 videos and their common structure
